Build Project Phases:

> compile
  to compile project (make classes in /target)
> test
  to test using JUnit
> package 
  to create .jar (war , ear) - depending on the type of project
> integration-test
  to start integration tests
> install
  to copy .jar (war , ear) to local repository
> deploy
  to publish file to remote repository

Special Commands:

> clean
  to delete all results of building
> site
  to make documentation
