Создание “Интернет-Банка”

Состоит из 2 приложений:
1. Веб-приложение интернет банк на основе Servlets + JSP + Hibernate

Роли пользователей в системе:
- “Главный банкир” может создавать специальные типы контрактов, которые потом будут устанавливаться для счетов пользователей
- “Главный менеджер” может управлять рабочими местами операторов
- “Оператор” может завести счет пользователя в системе и выпустить кредитную карту для пользователя 
- “Пользователь” может совершать переводы и пополнение счета.

Контракты, определяют комиссии которые будут сняты с пользователя и переведены на счет банка. Счет пользователя, хранит текущий баланс и все проводки совершаемые для данного счета. Обособленной частью системы будут являться финансовые транзакции, контролирующие любые операции пользователя.

2. Приложение веб-сервис для приема платежей пользователей будет разработан как Rest сервис. К данному сервису будет обращаться интернет-банк для перевода платежа на счет в другом банке.
